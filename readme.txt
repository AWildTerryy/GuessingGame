This is a simple guess the picture game using HTML CSS and Javascript. There are three difficulties and they are used to determine if the pictures are blured or not. The purpose of this is to use the Pixelator (https://github.com/AWildTerryy/Pixelator) and create a webapp that can be hosted by AWS.
